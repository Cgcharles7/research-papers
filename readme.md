# 📚 My Research Papers Library

Welcome! This repository serves as a personal archive and public library for various documents, research papers, and articles I've found interesting across the web.

## 🎯 Purpose
The goal of this project is to centralize high-quality information into one accessible location. Instead of hunting through bookmarks or various sources, I (and anyone else) can come here to find and read these materials easily.

## 📖 How to Use This Library
* **Browse:** Explore the folders above to see different categories.
* **Read:** Click on any `.pdf` file to view it directly in your browser via GitHub's built-in viewer.
* **Search:** Press `t` on your keyboard while on the main page to quickly search for a specific document title.
